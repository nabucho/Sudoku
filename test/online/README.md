Online technique fixtures
=========================

These `.sdkc` files are candidate-grid snapshots transcribed from public
technique examples. They complement the generated synthetic fixtures in
`test/synthetic/` by checking the solver against examples documented outside
this repository.

Each fixture records a `source_url` and keeps the same format as the synthetic
fixtures: expected placements/eliminations plus a 9x9 candidate grid. HoDoKu
examples are by Bernhard Hobiger and are published under the GNU FDLv1.3 on
the linked pages.
