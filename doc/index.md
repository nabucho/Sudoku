# Technique Documentation

This folder contains one Markdown page for each solver-facing logical technique implemented by the project.

The descriptions are original summaries based on established Sudoku references, especially HoDoKu, SudokuWiki, and Sudopedia. Where several implemented techniques are variants of the same family, the pages share common vocabulary so they stay consistent with `README.md`, inline docstrings, and progress-step output.
